### 2:1 布局
#### 伪元素
```html
<style>
    .container {
        background-color: #00acec;
    }
    .container::before{
        content: '';
        float: left;
        padding-bottom: 50%;
    }
    .container::after{
        clear: both;
        content: '';
        display: table;
    }
</style>

<div class="container">
```
#### 内容绝对定位
```html
<style>
    .container {
        background-color: #00acec;
        overflow: hidden;
        padding-top: 50%;
        position: relative;
        height: 0;
    }
    .content{
        position: absolute;
        top: 0;
        left: 0;
        right: 0;
        bottom: 0;
    }

</style>

<div class="container">
    <div class="content"></div>
</div>
```
#### 视窗单位vw/vh
```html
<style>
    .container {
        background-color: #00acec;
        width: 100vw;
        height: 50vw;
    }
</style>

<div class="container">
</div>
```

### 多列等高布局
1. `padding+margin+overflow`实现多列等高，兼容性好
2. border实现多列等高，左列浮动，伪元素清除浮动
3. display:flex多列等高
4. display:grid实现多列等高
5. display:table-cell实现多列等高
6. 父元素线性渐变背景色实现多列等高
```html
<div class="container">
    <div class="left">content<br/></div>
    <div class="right">
        content<br/>
        content<br/>
        content<br/>
        content<br/>
    </div>
</div>
```

```css
.container {
    position: relative;
    overflow: hidden;
    width: 1000px;
    color: pink;
}
.left {
    float: left;
    width: 200px;
    background-color: red;
    padding-bottom: 9999px;
    margin-bottom: -9999px;
}

.right {
    float: left;
    width: 800px;
    background-color: greenyellow;
    padding-bottom: 9999px;
    margin-bottom: -9999px;
}
```


```css
.container {
    position: relative;
    width: 800px;
    border-left: 200px solid purple;
    color: pink;
}
.container:after{
    content: '';
    display: block;
    height: 0;
    clear: both;
}

.left {
    float: left;
    width: 200px;
    margin-left: -200px;
}

.right {
    width: 800px;
    background-color: greenyellow;
}
```

```css
 .container {
    color: pink;
    display: flex;
    flex-direction: row;
    flex-wrap: nowrap;
    align-items: stretch;
}

.left {

    background-color: green;
}

.right {

    background-color: greenyellow;

```
